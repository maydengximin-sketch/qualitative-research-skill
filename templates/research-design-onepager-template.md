# 研究设计一页纸模板

> 选题/开题阶段用，快速把质性研究骨架搭起来。填完即可对照各 reference 深化。强调范式—方法—质量标准三者自洽（见 `references/13-faq-pitfalls.md`）。

| 要素 | 你的填写 |
| --- | --- |
| 研究问题（怎么样/为什么） | ＿＿＿ |
| 研究范式（建构/后实证/批判/实用） | ＿＿＿ |
| 方法学（扎根/现象学/个案/民族志/话语/内容分析） | ＿＿＿ |
| 数据来源与方法（访谈 N 人 / 文本 N 篇 / 政策 N 份 / 网络文本 N 条） | ＿＿＿ |
| 抽样策略（目的/理论/最大差异/滚雪球） | ＿＿＿ |
| 编码方案（开放-主轴-选择 / 主题六步 / 内容分析类目） | ＿＿＿ |
| 质量标准（按范式：信实度四标准 / Kappa / 三角验证 / 反身性） | ＿＿＿ |
| 饱和判定方式 | ＿＿＿ |
| 伦理（知情同意/匿名/数据安全/IRB） | ＿＿＿ |
| 预期成果（理论模型 / 主题框架 / 政策建议） | ＿＿＿ |

## 自洽性检查（三必须一致）
- [ ] 范式 ↔ 方法学匹配
- [ ] 范式 ↔ 质量标准匹配（建构主义→信实度；后实证/内容分析→Kappa）
- [ ] 分析深度 ↔ 目标匹配（描述→主题/内容；建理论→完整扎根）

## 选题三原则
小而深 · 有数据可得 · 有理论可挂靠。
