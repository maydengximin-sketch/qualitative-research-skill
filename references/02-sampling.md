# 02 · 抽样：目的抽样、理论抽样与样本量

> 质性研究**不追求统计代表性，追求信息丰富度（information power）**。抽样要为研究问题服务，能被论证，能被复现描述。

## 一、目的抽样的常见类型（Patton）

| 类型 | 逻辑 | 适用 |
| --- | --- | --- |
| 典型抽样 Typical | 选"普通/有代表性"的个案 | 描述常态现象 |
| 最大差异抽样 Maximum variation | 跨关键维度选差异最大的个案 | 找跨情境共通的主题 + 呈现多样性 |
| 极端/偏离个案 Extreme/deviant | 选最突出/异常的个案 | 研究成功/失败的极端 |
| 关键个案 Critical case | "若此处成立则普遍成立" | 逻辑推广 |
| 同质抽样 Homogeneous | 选高度相似者 | 焦点小组、深入某一亚群 |
| 滚雪球 Snowball | 由受访者引荐 | 隐蔽/难触及人群 |
| 标准抽样 Criterion | 满足预设标准者全纳入 | 质量保证、评价研究 |
| **理论抽样 Theoretical** | 由**已浮现的理论/范畴**决定下一个抽谁 | 扎根理论专用（见下） |

## 二、理论抽样（扎根理论核心，别和目的抽样混）

- **不是研究开始时定好的**，而是**编码过程中**根据正在浮现的范畴决定"接下来去找谁/什么资料，才能发展/检验这个范畴的属性与维度"。
- 流程：初始抽样（目的抽样起步）→ 编码出范畴 → 针对范畴的空白/边界做理论抽样 → 直到**理论饱和**。
- 写作时要展示这条链：每一轮抽样如何被上一轮分析驱动。

## 三、样本量：怎么定、怎么答审稿人

质性没有固定公式，但有可引用的经验区间（说明这是**起点估计**，最终以饱和为准）：

| 设计 | 常见区间 | 来源/依据 |
| --- | --- | --- |
| 现象学 | 5–25 人（IPA 常 3–10） | Creswell；Smith et al. |
| 扎根理论 | 20–30 人（至饱和） | Creswell；Morse |
| 个案研究 | 1 个深案 或 4–10 个跨案 | Yin；Stake |
| 民族志 | 1 个文化共享群体，长期 | — |
| 焦点小组 | 3–6 组 ×（6–10 人/组） | Krueger & Casey |
| 一般访谈饱和 | 常在 9–17 次访谈出现饱和 | Hennink & Kaiser (2022) 系统综述 |

**信息丰富度（information power, Malterud 2016）** 框架——样本越能满足以下条件，所需 N 越小：
1. 研究目标越**窄**；2. 样本特异性越**高**；3. 有无既有理论支撑；4. 对话质量越**高**；5. 分析策略是个案深析还是跨案。

> 答审稿人模板：先报实际 N 与抽样标准 → 说明用信息丰富度/饱和而非固定数确定 → 给出饱和判定证据（见 `06`）。**不要只说"达到饱和"而无证据。**

## 四、抽样论证段落（可套用结构）

1. 抽样策略名称 + 为何契合 RQ 与方法学
2. 纳入/排除标准（criterion）
3. 招募渠道与过程
4. 最终样本 N 与关键特征（用表呈现人口学/情境变量，匿名化）
5. 样本量如何确定（信息丰富度 / 饱和）
6. （扎根理论）理论抽样如何随分析推进
