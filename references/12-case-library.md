# 12 · 案例实战库（按数据类型）

> 四类典型研究的"研究问题 → 数据 → 方法 → 编码框架 → 结论"完整链路，帮用户把方法落到自己的题目上。案例为示意性骨架，供照着搭建。

## 一、访谈类 · 扎根理论：教师专业认同
| 环节 | 内容 |
| --- | --- |
| 研究问题 | 新手教师在职业生涯早期，专业认同如何形成？ |
| 数据 | 目的+滚雪球抽样，深度访谈 15 位入职 1–5 年教师，逐字转录 |
| 方法 | 扎根理论三级编码 + 双人编码 Kappa |
| 开放编码 | ~80 个初始概念（"第一次被学生认可""适应应试压力"…） |
| 主轴编码 | 6 主范畴：入职冲击、关键事件、支持系统、自我调适、价值认同、未来规划 |
| 选择编码 | 核心范畴"专业认同的动态建构"，串"冲击—调适—认同"故事线 |
| 信度 | 第二人独立编码 Kappa 0.74–0.88 |
| 结论 | 三阶段建构模型 + 师徒制等建议 |

可类比：孤独症儿童父母创伤后成长、护理人员职业体验、运动员持续参与动机。

## 二、政策文本类 · 内容分析：教育数字化政策
| 环节 | 内容 |
| --- | --- |
| 研究问题 | 我国基础教育数字化政策在政策工具使用上的结构特征？ |
| 数据 | 权威渠道检索国家级政策文件 N 份，登记机关/年份/文号 |
| 方法 | 内容分析 + 政策工具（供给/环境/需求）× 主题 二维编码（见 `08`） |
| 分析 | 矩阵编码查询出交叉频次，统计各工具占比 |
| 结论 | 供给型偏多、需求型不足，结构性失衡；建议加强需求侧 |

可类比：劳动教育政策、数字政府、体育产业政策、中美政策对比。

## 三、网络文本类 · 主题+话语分析：小红书形象建构
| 环节 | 内容 |
| --- | --- |
| 研究问题 | 小红书笔记如何建构"北欧"的地区形象？ |
| 数据 | 关键词检索筛选笔记，采集正文+热度+配图，匿名化（见 `09`） |
| 方法 | 主题分析 + 话语分析 + 词频/词云 + 配图多模态 |
| 编码框架 | 自然环境、社会福利、生活方式、审美想象、文化符号 |
| 结论 | 北欧被建构为"理想生活"的投射对象，兼具向往与想象性美化 |

可类比：大众点评消费体验、品牌联名态度、虚拟偶像/粉丝、旅游游记。

## 四、多模态类 · 主题分析：课堂实录中的素养
| 环节 | 内容 |
| --- | --- |
| 研究问题 | 音乐师范生核心素养在教学实录中如何体现？ |
| 数据 | 课堂实录视频 + 访谈录音，导入 NVivo（链接方式），关联转录稿 |
| 方法 | 时间轴片段编码 + 转录稿主题分析 |
| 编码框架 | 专业知识、教学技能、育人意识、反思能力 |
| 结论 | 提炼素养评价指标体系，支撑培养方案改进 |

## 用法
让用户选与自己最接近的一类，照"研究问题→数据→方法→编码框架→结论"五栏填自己的内容，即得研究骨架；再到对应 reference 深化。
