图1 · 扎根理论操作循环（理论抽样 → 饱和）

扎根理论区别于一般编码的核心：收集与分析交替，由浮现的范畴驱动下一轮抽样，直到理论饱和。

flowchart TD A[确定研究问题
过程/机制取向] --> B[初始抽样
目的抽样起步] B --> C[收集数据
访谈/观察/文档] C --> D[开放编码
逐行贴标签·实境码] D --> E[持续比较
事件↔事件 码↔码] E --> F[主轴编码
归并范畴·建立关系] F --> G[写理论备忘录] G --> H{理论饱和?
不再产生新范畴/属性} H -- 否 --> I[理论抽样
针对范畴空白补数据] I --> C H -- 是 --> J[选择性编码
确定核心范畴] J --> K[提炼故事线·构建理论] K --> L[输出概念模型图 + 命题] style H fill:#fff3cd,stroke:#b7791f style L fill:#d1f0e0,stroke:#138a5b style G fill:#e7efff,stroke:#2563eb

图2 · 三级编码漏斗（数据 → 理论）

从贴近数据的初始概念，逐层抽象到核心范畴与理论。

flowchart TD subgraph L1[开放编码 · 贴近数据] a1[原始语句1] --> c1[初始概念] a2[原始语句2] --> c1 a3[原始语句3] --> c2[初始概念] a4[原始语句4] --> c3[实境码] end subgraph L2[主轴编码 · 建立关系] c1 --> cat1[范畴 A] c2 --> cat1 c3 --> cat2[范畴 B] end subgraph L3[选择性编码 · 整合理论] cat1 --> core[核心范畴] cat2 --> core core --> theory[扎根理论/模型] end style core fill:#fff3cd,stroke:#b7791f style theory fill:#d1f0e0,stroke:#138a5b

图3 · 三流派编码语言对照

标题与正文必须用同一流派，切忌混用。

flowchart LR subgraph G[经典 Glaser] g1[实质编码] --> g2[理论编码] end subgraph S[程序化 Strauss & Corbin] s1[开放编码] --> s2[主轴编码] --> s3[选择性编码] end subgraph C[建构主义 Charmaz] c1[初始编码] --> c2[聚焦编码] --> c3[理论编码] end

图4 · 编码范式（主轴编码）

flowchart LR A[因果条件] --> P((现象)) B[脉络/情境] --> P C[中介条件] --> P P --> D[行动/互动策略] D --> E[结果] style P fill:#fff3cd,stroke:#b7791f style E fill:#d1f0e0,stroke:#138a5b

图5 · 理论模型结构模板（填空即用）

替换为你的核心范畴与主范畴，即得论文里的理论模型图。

flowchart TD COND[前因条件
主范畴①②] --> CORE{核心范畴
理论中心} CONTEXT[情境/脉络
主范畴③] -. 调节 .-> CORE CORE --> STRAT[行动策略
主范畴④⑤] STRAT --> OUTCOME[结果/效应
主范畴⑥] OUTCOME -. 反馈 .-> COND style CORE fill:#fff3cd,stroke:#b7791f,stroke-width:2px style OUTCOME fill:#d1f0e0,stroke:#138a5b style COND fill:#e7efff,stroke:#2563eb
模型图箭头要对应数据支撑的命题，并在正文逐条论证。

图1 · Braun & Clarke 反思性主题分析六步

不是直线流程：第4步检验主题常需回到第2/3步反复修订。

flowchart TD S1[1. 熟悉数据
反复阅读·记想法] --> S2[2. 生成初始编码
系统覆盖全数据] S2 --> S3[3. 生成候选主题
编码聚类成模式] S3 --> S4{4. 检验主题
对照编码片段+整个数据集} S4 -- 需重组 --> S2 S4 -- 通过 --> S5[5. 定义与命名
写主题故事+精炼名] S5 --> S6[6. 撰写报告
引语证据+分析叙述] S6 --> ANS[回应研究问题] style S4 fill:#fff3cd,stroke:#b7791f style S6 fill:#e7efff,stroke:#2563eb style ANS fill:#d1f0e0,stroke:#138a5b
